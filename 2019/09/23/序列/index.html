#序列

定义

1. 序列是Python中的一种数据结构，用于保存一组有序数据。
2. 所有的数据在序列中都有一个唯一的索引,即可以用s[0]来表示
3.  会按照添加顺序来分配索引

序列的分类

1. 可变序列：列表（list）
2. 不可变序列：字符串（String）、元组（tuple）

序列的通用操作

1. 索引：s[index]，可以从前向后开始，从0开始(0,1,2)，也可以从后向前索引，从-1开始(-1,-2)
2. 切片：从列表中获取一个子列表，语法：列表[开始位置:结束位置:步长]；开始位置和结束位置都可以
        省略默认为第一个和最后一个，步长也可省略 默认为1，不可以为0，为负数时，代表从后向前取值。

函数：不需要对象调用，相当于Python全局方法；方法：需要相应Python对象去调用s.x()

常用操作函数 + *

1. + ： 用在两个序列中，将两个序列进行拼接
2. * ： 将一个序列复制指定次数
3. in/not in 判断元素是否在指定序列中 s in sequence
4. min()/max() 返回序列中最小/大值 min(sequence)、max(sequence)
5. list() 返回序列中的列表

常用方法

1. s.index(obj) 返回指定元素在序列中的位置，如果不存在，则抛出ValueError
2. s.index(obj1 [,startIndex, endIndex]) 返回某元素在序列中的索引
3. s.count(obj) 统计指定元素在列表中出现的次数
4. s.insert(index, obj) 用于可变序列中，向指定索引位置插入新元素，其后元素依次往后排
5. del s[index] 用于可变序列中，删除指定元素，其后元素依次向前排

列表（list）

列表的定义

Python中的一个对象 有序（按照添加顺序） 可以存放不同同类型的数据

列表的语法

1. myList = []; #定义一个空列表
2. myList = list(sequence); #将一个序列转化为一个列表

元组（tuple）

元组的定义

Python中的一个对象 不可变对象，当希望数据不可变时使用，其他情况都用列表

元组的语法

1. myTuple = (); #定义一个空元组
2. myTuple = (1, 'str'); #元组不可变，不能为元素重新赋值
3. myTuple = 1,2,3; #当元组不为空时，元组的小括号可以省略
4. myTuple = 1, ; #当括号省略时，至少要保留一个逗号
5. a,b,c = myTuple; #元组的解包，将元组的元素依次赋值给变量a b c,必须保证元素个数一致
6. a,b,*c = myTuple; # 当变量带*时，将非带*的变量赋值，剩余元素赋值给带*变量
7. 解包适合于其他任何数据类型

Python中的其他数据结构

字典（dict）

字典的定义

1. 字典属于一种新的数据结构，称为映射，类似Java中的Map。
2. 字典的值可以是任意对象，字典的键可以是任意的不可变对象

字典的语法

1. myDict = {}; #定义一个空字典
2. myDict = {'name':'alan', 'age':18}; # 定义一个非空字典
3. myDict = dict(name='alan', age=18); # 使用dict()函数来创建字典，参数为双值序列
4. myDict['name'] ; #根据键来获取值，为空时抛出ValueError，键的表示方法可以使用变量
5. myDict.get(key[, default]] ; #获取指定键对应的key,无对应值是返回default;
6. len(myDict) ; # 返回字典中的键值对个数
7. myDict.update([other]); # 将other中元素添加到当前字典中，如果重复，则以参数中的元素为主
8. myDict.keys() myDict.values() myDict.items();遍历当前字典

集合（set）

集合的定义

1. 只能存储不可变对象
2. 无序且不可重复

集合的语法

1. mySet = {1,2}; # 集合使用{}来创建非空集合，以区别和空字典的定义
2. mySet = set(); # 空集合只能使用函数set()来创建
3. mySet = set(sequence/dict); #使用函数set()将序列和字典转化为集合, 字典转化时，只会取其键
4. len(mySet); #使用函数len() 获取集合的长度
5. mySet.add(e); #使用函数向集合中添加元素
6. mySet.update([other]); #将两个集合合并到当前集合中
7. mySet.pop(); #删除第一个元素，因元素的内容无序，故无法区别删除那个元素
8. mySet.remove(e); #删除集合中指定元素， 当元素不存在时，抛出KeyError
9. &、|、-、^、<=、<； #分别求两个集合的交集，并集、差集、异或集、子集、真子集
